# Markdown Viewerへようこそ

このサイトはReact + TypeScriptで作成されたMarkdownビューアーです。

## 機能

- **Markdownレンダリング**: Markdownファイルを美しく表示
- **ルーティング**: React Routerによる複数ページ対応
- **GitHub Flavored Markdown**: テーブルやタスクリストなどの拡張記法をサポート

## 使い方

左側のサイドバーからページを選択してください。各ページはMarkdownファイルから生成されています。

### サポートされている記法

- 見出し
- リスト（順序付き・順序なし）
- コードブロック
- テーブル
- リンク
- 画像
- 引用
- そして、もっと多くの機能！
